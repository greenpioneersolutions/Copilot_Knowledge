# Sample prompts

## Manager Analyst
- `Explain this repo to a manager who is new to the team.`
- `Summarize the current PRs and tell me where release risk may exist.`
- `Create a manager-readable technical brief from this codebase.`

## Delivery Planner
- `Create a delivery plan for this feature request.`
- `Break this work into smaller reviewable issues.`
- `Turn this rough ask into acceptance criteria and rollout steps.`

## CLI
- `Give me an overview of this project.`
- `@README.md summarize setup risk and dependencies`
- `/research Compare the stated rollout process with likely weak spots in this repo`
