# When to use each agent

Only two shared agents are introduced in Level 2.

That is intentional.

## 1) Manager Analyst

### Use when
- you need a repo, issue, or PR explained in manager language
- you need a status draft
- you need a technical brief
- you need likely risks or next questions

### Do not use when
- you need code edits
- you need to break work into implementation steps
- you need delegation to an engineering agent

## 2) Delivery Planner

### Use when
- you need a plan
- you need work broken down
- you need acceptance criteria
- you need rollout and validation steps
- you need risks, dependencies, and open questions surfaced

### Do not use when
- you just need a quick explanation
- you want a stakeholder-friendly communication draft
- the ask is too vague to plan

## Default order

1. Use **Manager Analyst** to understand the context.
2. Hand off to **Delivery Planner** to turn that context into a plan.
