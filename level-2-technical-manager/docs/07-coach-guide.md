# Coach guide

## Suggested format

- 10 minutes — why VS Code now
- 15 minutes — Ask and Plan mode demo
- 15 minutes — use Manager Analyst and Delivery Planner on real work
- 10 minutes — light CLI inside VS Code terminal
- 10 minutes — each learner produces one plan or one technical brief

## What the coach should emphasize

- Ask or Plan first
- agents are shared personas, not magic
- use the smallest useful tool
- keep CLI light
- always review

## What the coach should avoid

- do not introduce more than two custom agents
- do not introduce coding agent yet
- do not introduce MCP
- do not introduce a large prompt library
