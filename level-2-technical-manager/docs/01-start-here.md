# Start here

Level 2 adds only enough power to materially improve output quality.

## What changes from Level 1

You now add:
- VS Code
- workspace and file context
- built-in chat modes
- two custom agents
- light terminal usage

## What does not change

You still:
- review before posting
- prefer simple workflows
- use a shared Space and repo context
- avoid teaching every feature

## The core idea

Level 2 is not about "becoming technical enough for Copilot."

It is about:

> using the real codebase and repository context to produce better plans, better summaries, and better manager decisions
