# Level 2 — Work With It

This repo is the **technical manager lane**.

Level 2 starts only after a learner is comfortable with Level 1.

## What problem this level solves

Level 2 solves the question:

> "How do I get better repo-grounded answers and better planning output without turning into a full-time toolsmith?"

The answer is:

- add VS Code
- use built-in chat modes
- use two shared agents
- use a small amount of CLI inside the VS Code terminal

## What the learner gets

### Surfaces
- VS Code
- GitHub.com
- integrated terminal

### Built-in modes to learn
- Ask
- Plan
- Edit

### Shared custom agents
- Manager Analyst
- Delivery Planner

### Light CLI moves
- `copilot`
- `@path/to/file`
- `/research TOPIC`
- `/model`

## What stays out of scope

Not in Level 2:
- big agent catalogs
- MCP setup
- advanced hooks
- coding-agent-first workflows
- prompt-file sprawl

## Who this level is for

Use Level 2 for:
- technical managers
- managers who need deeper repo context
- managers who draft plans, rollouts, and technical briefs
- champions who are not yet ready to maintain a full manager AI operating system

## Success signal

A Level 2 learner can:
- use Ask or Plan mode effectively in VS Code
- use one of the two shared agents without confusion
- use the CLI to get a repo-grounded answer or research brief
- turn repo context into a stronger plan or status update
