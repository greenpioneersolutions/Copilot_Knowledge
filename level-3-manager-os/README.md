# Level 3 — Scale It

This repo is the **manager operating system lane**.

Level 3 is for champions, senior managers, directors, and technical leaders who want repeatable team workflows without drowning people in agent sprawl.

## What problem this level solves

Level 3 solves the question:

> "How do we standardize the highest-friction manager workflows, reduce communication overhead, capture decisions, and delegate bounded work safely?"

The answer is:

- keep the shared catalog small
- use four custom agents only
- add the built-in Copilot coding agent as the fifth agent in the workflow
- focus on a few high-value manager jobs

## The five agents in the Level 3 system

### Four custom agents in this repo
1. Manager Analyst
2. Delivery Planner
3. Communications Writer
4. Decision Logger

### One built-in agent
5. Copilot coding agent

## Why only five agents

The point of Level 3 is not "give managers every possible persona."

The point is:
- fewer choices
- clearer use cases
- repeatable team behavior
- easier maintenance

## What this level is for

Use Level 3 for:
- AI champions
- directors and senior managers
- platform or enablement owners
- managers who want to teach others
- teams ready to delegate well-scoped work

## Success signal

A Level 3 user can:
- pick the right agent quickly
- move from context → plan → communication → decision capture
- delegate bounded engineering work using a strong issue
- review and iterate on AI output safely
- maintain a small catalog without creating confusion
