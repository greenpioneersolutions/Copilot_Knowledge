# Agent catalog

This is the entire Level 3 catalog.

## 1) Manager Analyst

### Use when
- you need repo, issue, or PR context explained for a manager
- you need a technical briefing
- you need likely risks or next questions

### Do not use when
- you need a delivery plan
- you need a stakeholder communication draft
- you need code edits

---

## 2) Delivery Planner

### Use when
- you need a scoped plan
- you need issue breakdown
- you need acceptance criteria
- you need rollout and validation steps
- you need risks and dependencies mapped

### Do not use when
- you just need a summary
- the ask is too vague to plan without more questions

---

## 3) Communications Writer

### Use when
- you need a status update
- you need release notes
- you need upward communication
- you need incident or stakeholder messaging grounded in repo reality

### Do not use when
- you have not understood the context yet
- you need a technical plan
- you need to capture a decision record

---

## 4) Decision Logger

### Use when
- you have meeting notes or a transcript
- you need a decision log
- you need an ADR-like artifact
- you need FAQ or onboarding memory from live discussion

### Do not use when
- you need a technical summary first
- you need a communication draft for leadership

---

## 5) Copilot coding agent (built-in)

### Use when
- the issue is clear
- acceptance criteria exist
- dependencies are visible
- the work is bounded enough to review
- you are willing to review the resulting PR

### Do not use when
- the request is ambiguous
- the work is strategic rather than bounded
- the issue is still missing assumptions or open questions
- nobody has time to review the output

## Recommended order

In most cases:
1. Manager Analyst
2. Delivery Planner
3. Communications Writer or Decision Logger
4. Copilot coding agent
