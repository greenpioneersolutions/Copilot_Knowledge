# Level 3 operating model

## What Level 3 is

Level 3 is not a prompt library.

It is a small set of repeatable workflows that match real manager time sinks.

## What Level 3 is not

It is not:
- every Copilot feature
- a giant catalog of clever agents
- full automation of management work
- a replacement for judgment

## The operating principles

### 1. Start with context
Use the repo, Space, issues, PRs, and documentation before drafting anything.

### 2. Prefer read-only agents
Read-only agents are safer and easier to trust.

### 3. Delegate only bounded work
Use the coding agent only when the issue is scoped well enough to review the result meaningfully.

### 4. Save useful output back to Markdown
If something matters tomorrow, it should not live only in chat history.

### 5. Keep the shared catalog tiny
Each new agent increases teaching, maintenance, and confusion cost.
