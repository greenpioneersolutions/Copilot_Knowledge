# Start here

Level 3 is where AI becomes a small operating system for manager work.

## The rule

Do not add more agents unless the current catalog is clearly insufficient.

## The workflow model

Most Level 3 work should follow this path:

1. **Understand** using Manager Analyst
2. **Plan** using Delivery Planner
3. **Communicate** using Communications Writer
4. **Capture decisions** using Decision Logger
5. **Delegate bounded code work** using the built-in Copilot coding agent

## What makes this safe

- the catalog is small
- most agents are read-only
- coding delegation comes last
- the team uses templates and review habits
- governance is explicit
