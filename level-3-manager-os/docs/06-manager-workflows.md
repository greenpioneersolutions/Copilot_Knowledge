# Manager workflows

## Workflow 1 — Weekly status pack

### Flow
Manager Analyst → Communications Writer

### Output
- what changed
- risks
- decisions needed
- support needed
- next checkpoint

## Workflow 2 — Meeting to durable memory

### Flow
Decision Logger

### Output
- decision record
- rationale
- alternatives considered
- follow-up owners
- FAQ entries if needed

### Flow
Manager Analyst → Delivery Planner → built-in coding agent

### Output
- cleaner issue
- acceptance criteria
- dependencies
- bounded delegated task
- reviewable PR

## Workflow 4 — PR to leadership narrative

### Flow
Manager Analyst → Communications Writer

### Output
- what changed
- why it matters
- user or operational impact
- risk note
- release callouts
