# Measures

## Good Level 3 measures

### Communication
- time to draft weekly status update
- time to draft release note or stakeholder update
- manager self-report on communication effort reduction

### Planning
- percentage of delegated issues that were well scoped on first pass
- issue quality score before vs after planner use
- review loop count for work that used the planner

### Memory and knowledge
- decision logs captured per month
- repeated question reduction
- onboarding usefulness of saved Markdown artifacts

### Delegation
- percentage of coding-agent tasks that produced a usable PR
- number of follow-up iterations needed
- review turnaround time
- cases where the issue needed major clarification after delegation

## What not to optimize for

Do not optimize Level 3 around:
- raw prompt volume
- number of agents created
- vanity usage charts
- PR throughput without quality context
