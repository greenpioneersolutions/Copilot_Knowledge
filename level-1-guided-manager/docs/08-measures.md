# Measures

Do not rely on GitHub usage dashboards alone for Level 1.

Level 1 is web-first and Space-heavy, so it can be undercounted in IDE-focused telemetry.

## Better Level 1 measures

### Adoption
- number of managers who completed a Level 1 session
- number of shared Spaces actually used in the past 2 weeks
- number of managers who used at least one of the three plays on real work

### Quality
- issue quality before vs after Level 1
- PR question quality before vs after Level 1
- manager confidence score after a real workflow

### Time saved
- time to create a clean issue
- time to understand a PR enough to ask useful questions
- repeated-question reduction reported by team leads

## Keep it lightweight

Use a simple pulse survey:
- I know where to start with Copilot.
- I can use it to shape work.
- I can use it to understand work.
- I trust myself to review the output before posting.
