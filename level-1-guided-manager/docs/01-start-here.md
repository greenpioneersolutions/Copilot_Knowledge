# Start here

This level is intentionally small.

## Your mission

Use GitHub Copilot in one place and get one useful result quickly.

## Where you work

- GitHub.com
- one shared Space
- repository, issue, and PR pages

## What you learn

### Play 1 — understand the work
Ask the shared Space to explain the repository or workstream in plain English.

### Play 2 — shape the work
Turn rough notes, screenshots, or meeting takeaways into a clean issue.

### Play 3 — review the work
Use PR summary and Copilot code review to understand what changed and what questions to ask.

## What you do not need to learn yet

You do not need to learn:
- VS Code
- file references
- terminal commands
- agents
- prompt files
- model pickers

That is deliberate. The goal is confidence first, complexity later.

## Your only rule

Do not post or submit anything without reviewing it.
