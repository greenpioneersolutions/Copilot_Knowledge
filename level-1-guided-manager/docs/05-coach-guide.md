# Coach guide

## Session shape

Run Level 1 as a short practical session.

### Suggested agenda

- 5 minutes — what this is and what it is not
- 10 minutes — demo the three plays
- 15 minutes — each learner tries one real issue or one real PR
- 5 minutes — review checklist and close

## What the coach should say repeatedly

- Start in the shared Space.
- Do not ask Copilot to do everything.
- Review before posting.
- If the answer is weak, give better context instead of giving up.

## What the coach should not do

- do not open VS Code
- do not explain models
- do not explain custom agents
- do not explain prompt files
- do not turn this into a product tour

## Coaching success signal

The learner should finish the session with one real useful output, not a generic demo.
