# Level 1 team Space manifest

Use this as a checklist when building the shared Space.

## Add these sources

- core repository
- one onboarding note
- one glossary note
- one current priorities note
- a few high-signal issues
- a few high-signal pull requests

## Do not add these yet

- giant PDF dumps
- stale reports
- low-value notes
- duplicate docs
- everything in the org

## Space purpose

This Space exists to help managers:
- understand the work
- shape the work
- review the work
