# Level 1 — Try It

This repo is the **beginner lane**.

It is designed for managers who are new to GitHub Copilot or who felt overwhelmed by a more technical setup.

## What problem this level solves

Level 1 solves the question:

> "How do I get value from GitHub Copilot without learning VS Code, CLI, models, prompt files, or custom agents?"

The answer is:

- use GitHub.com
- use one shared Space
- use three simple workflows
- always review before posting

## What the learner gets

### One surface
GitHub.com only.

### One memory model
One shared Space per team or workstream.

### Three workflows
1. understand the work
2. turn rough notes into a clean issue
3. understand a pull request as a manager

### One habit
Review before posting.

## What stays out of scope

Not in Level 1:
- VS Code
- CLI
- prompt files
- custom agents
- coding agent
- MCP
- model selection

## Who this level is for

Use Level 1 for:
- managers who mostly use GitHub.com
- non-technical or less technical managers
- leaders who need engineering context without editor complexity
- first-time Copilot users

## Success signal

A Level 1 learner can do these three things in one sitting:
- ask a Space to explain a system or workstream
- turn notes into a good issue
- read a PR summary and ask better questions

## Admin setup at a glance

- create one shared Space
- keep the Space small and curated
- add one short repository instruction file
- give learners the three plays and the review checklist
